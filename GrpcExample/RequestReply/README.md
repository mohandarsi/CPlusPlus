# gRPC Request Reply

Simple client & server example by using [gRPC]
(https://grpc.io/).


1) Cmake & Vcpkg & gRPC.  
2) simple request and reply by using RPC.

## Build

Use Visual stuido 2019 and CMake
Install [vcpkg](https://vcpkg.io/en/getting-started.html)  


```bash
vcpkg install gRPC
```
